# AskMo
June 2026
Designed and developed a responsive, single-page web application serving as an interactive dining and navigation guide for Hartsfield-Jackson Atlanta International Airport. The app features a chat interface for querying 154+ restaurants across 8 concourses, an interactive terminal map with gate-level wayfinding, a searchable and filterable restaurant directory, and a live flight tracker integrated with the AviationStack API. Restaurant data is loaded dynamically via Excel file upload using the SheetJS library, which allows the website to be used without artificial intelligence. Built entirely with vanilla HTML, CSS, and JavaScript; optimized for mobile with dynamic viewport units. It was deployed using GitHub Pages.
